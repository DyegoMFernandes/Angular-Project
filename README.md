# Full Stack Project - Angular & C#

This is a full-stack application developed to practice and demonstrate skills in modern web development.

The project consists of:

- A RESTful API built with C# and ASP.NET Core.
- A frontend application developed with Angular.
- Integration between frontend and backend through HTTP requests.
- Data management following common software development best practices.

## Technologies Used

### Backend
- C#
- ASP.NET Core
- Entity Framework Core
- SQL Database

### Frontend
- Angular
- HTML
- TailWindCSS

## Features

- CRUD operations through a REST API.
- Data consumption and presentation in Angular.
- Form handling and validation.
- Routing and component-based architecture.

## Purpose

The main objective of this project is to strengthen knowledge in full-stack development, focusing on API creation with ASP.NET Core and frontend development with Angular.
